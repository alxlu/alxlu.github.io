More Melody



More Melody was a schoool project. We had to create a performance to a song by a band named Lucky Dragons. I ended up with a song called More Melody. Because the song reminded me of coins being shaken in a jar, I designed my performance to resemble that.

The main screen displays a black monolith that shows up. When the song starts, the monolith becomes clear and the internals--a lot of black spheres are shown. If a user visits the website specified during the performance using a device with a webcam, their image is automatically mapped to a sphere and is put into the monolith.

Everything is done via a web browser

Technology

More Melody uses WebRTC to display video streams from participating audience members and WebGL for the 3D graphics.
