Work from Pixar Animation Studios Internship

Some work from when I was an intern in Pixar's Undergraduate Program (PUP).


Jessie Paint

Original scene was done using Maya particles that were processed in Houdini. This is my attempt at doing it using Houdini exclusively. Done at Pixar Animation Studios under mentorship of FX Technical Director Matt Wong.

Building Explosion

Created using Houdini's PyroFX and Pixar's in-house rigid body simulator.

The up-res version had problems with the auto-resizing fluid container I had set up; the container did not resize fast enough (this was before pyroFX automatically resized in in Houdini 12). Did not have time to do a re-sim with the fixed resize setup.
